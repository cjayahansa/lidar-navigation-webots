## Lidar Sensor Using Navigation (Webots)

This project is a learning-based robotics simulation built in **Webots** using **Python**.
The main goal of this project is to understand how a **LiDAR sensor** works and how it can be used for **robot navigation** and **obstacle detection**.

In this simulation, the robot uses a LiDAR sensor to detect the distance to nearby objects and navigate safely without collisions. This helps to simulate real-world applications like autonomous robots, self-driving cars, and indoor mapping systems.

---

###  Features

* Reads LiDAR data in real time using Webots.
* Displays distance measurements from the sensor.
* Basic robot movement and navigation control.
* Detects obstacles and avoids collisions automatically.
* Written in **Python** with clear and simple logic for beginners.
